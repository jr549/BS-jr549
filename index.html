Responsive Data Integration

This purpose of this site is to demonstrate in a real world scenario responsive and seemless data integration using weather data acquired over the web from a remote server via Json. I used the APIs of AngularJS to fetch and bind the data with Bootstrap to adapt the page contents to various devices and screen sizes.
